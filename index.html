Geo-Facial AR Explorer

This project is a web-based AR application that synthesizes real-time GPS and compass data with Leaflet.js mapping and FaceAPI.js facial recognition to create an immersive, location-aware interface. Users can explore their surroundings through an AR view that overlays geospatial information and facial recognition insights, providing a unique blend of navigation and interactive technology.

Face & Expression Detection

This project focuses on real-time biometric analysis directly in the browser. By leveraging FaceAPI.js (built on TensorFlow.js), the application can identify human faces, map their landmarks, and interpret emotional states without needing a backend server.
